# Supply Chain Analytics Backend\n\nComplete FastAPI-based supply chain analytics platform with automated order lifecycle tracking, SLA monitoring, and bottleneck detection.\n\n## Features\n\n### Core Analytics Engine\n- **Automated Time Calculation**: Calculates procurement, processing, dispatch, and delivery durations\n- **SLA Validation**: Monitors SLA compliance with configurable thresholds\n- **Bottleneck Detection**: Identifies stages causing maximum delays\n- **Real-time Analytics**: Aggregate analytics and performance metrics\n\n### API Capabilities\n- Order lifecycle management with automated analytics\n- Product and supplier management\n- Analytics dashboards and reports\n- RESTful API with comprehensive endpoints\n- Swagger/OpenAPI documentation\n\n## Project Structure\n\n```\nsmart-retail-backend/\n├── app/\n│   ├── routes/          # API endpoints\n│   ├── models/          # SQLAlchemy database models\n│   ├── schemas/         # Pydantic schemas\n│   ├── services/        # Business logic layer\n│   ├── utils/           # Utility functions\n│   ├── middleware/      # Middleware components\n│   └── database/        # Database configuration\n├── ml/                  # Machine learning modules (future)\n├── uploads/             # File upload storage\n├── main.py              # FastAPI application\n├── requirements.txt     # Python dependencies\n└── .env                 # Environment configuration\n```\n\n## Installation\n\n### Prerequisites\n- Python 3.9+\n- pip\n\n### Setup\n\n1. **Install dependencies**:\n```bash\ncd smart-retail-backend\npip install -r requirements.txt\n```\n\n2. **Configure environment**:\nEdit `.env` file with your settings:\n```env\nDATABASE_URL=sqlite:///./supply_chain_analytics.db\nAPI_HOST=0.0.0.0\nAPI_PORT=8000\n```\n\n3. **Run the application**:\n```bash\npython main.py\n```\n\nOr with uvicorn:\n```bash\nuvicorn main:app --reload --host 0.0.0.0 --port 8000\n```\n\n4. **Access API documentation**:\n- Swagger UI: http://localhost:8000/docs\n- ReDoc: http://localhost:8000/redoc\n\n## API Endpoints\n\n### Orders (`/api/orders`)\n- `POST /` - Create order with automated analytics\n- `GET /` - List all orders\n- `GET /{order_id}` - Get order by ID\n- `GET /by-number/{order_number}` - Get order by number\n- `GET /status/{status}` - Get orders by status\n- `GET /analytics/sla-breaches` - Get SLA breach orders\n- `GET /analytics/bottleneck/{stage}` - Get orders by bottleneck\n- `GET /analytics/summary` - Get analytics summary\n- `PATCH /{order_id}/status` - Update order status\n- `DELETE /{order_id}` - Delete order\n\n### Products (`/api/products`)\n- `POST /` - Create product\n- `GET /` - List all products\n- `GET /{product_id}` - Get product by ID\n- `GET /sku/{sku}` - Get product by SKU\n- `GET /category/{category}` - Get products by category\n- `DELETE /{product_id}` - Delete product\n\n### Suppliers (`/api/suppliers`)\n- `POST /` - Create supplier\n- `GET /` - List all suppliers\n- `GET /{supplier_id}` - Get supplier by ID\n- `DELETE /{supplier_id}` - Delete supplier\n\n### Analytics (`/api/forecast`)\n- `GET /overview` - Overall analytics overview\n- `GET /bottleneck-analysis` - Detailed bottleneck analysis\n- `GET /sla-compliance` - SLA compliance metrics\n\n### Authentication (`/api/auth`)\n- `POST /register` - Register new user\n- `POST /login` - User login\n- `GET /users/{user_id}` - Get user details\n\n## Usage Example\n\n### Creating an Order with Analytics\n\n```python\nimport requests\nfrom datetime import datetime, timedelta\n\n# Prepare order data\norder_data = {\n    \"order_number\": \"ORD-2026-0001\",\n    \"product_id\": 1,\n    \"supplier_id\": 1,\n    \"quantity\": 100,\n    \"unit_price\": 50.00,\n    \"order_placed_at\": datetime.now().isoformat(),\n    \"procurement_completed_at\": (datetime.now() + timedelta(hours=30)).isoformat(),\n    \"processing_completed_at\": (datetime.now() + timedelta(hours=50)).isoformat(),\n    \"dispatched_at\": (datetime.now() + timedelta(hours=60)).isoformat(),\n    \"delivered_at\": (datetime.now() + timedelta(hours=120)).isoformat(),\n    \"status\": \"delivered\"\n}\n\n# Create order\nresponse = requests.post(\"http://localhost:8000/api/orders/\", json=order_data)\norder = response.json()\n\nprint(f\"Order created with analytics:\")\nprint(f\"- Procurement Time: {order['procurement_time']} hours\")\nprint(f\"- Processing Time: {order['processing_time']} hours\")\nprint(f\"- Total Time: {order['total_time']} hours\")\nprint(f\"- SLA Breach: {order['sla_breach']}\")\nprint(f\"- Bottleneck Stage: {order['bottleneck_stage']}\")\n```\n\n### Getting Analytics Summary\n\n```python\nresponse = requests.get(\"http://localhost:8000/api/orders/analytics/summary\")\nanalytics = response.json()\n\nprint(f\"Analytics Summary:\")\nprint(f\"- Total Orders: {analytics['total_orders']}\")\nprint(f\"- SLA Compliance Rate: {analytics['sla_compliance_rate']}%\")\nprint(f\"- Average Total Time: {analytics['avg_total_time']} hours\")\nprint(f\"- Bottleneck Summary: {analytics['bottleneck_summary']}\")\n```\n\n## Analytics Features\n\n### Duration Calculations\nAutomatically calculates:\n- **Procurement Time**: Order placement → Procurement completion\n- **Processing Time**: Procurement → Processing completion\n- **Dispatch Time**: Processing → Dispatch\n- **Delivery Time**: Dispatch → Delivery\n- **Total Time**: Order placement → Delivery\n\n### SLA Thresholds (Configurable in .env)\n- Procurement: 48 hours\n- Processing: 24 hours\n- Dispatch: 12 hours\n- Delivery: 72 hours\n- Total: 156 hours\n\n### Bottleneck Detection\nIdentifies the stage with maximum delay:\n- Procurement\n- Processing\n- Dispatch\n- Delivery\n\n## Database Schema\n\n### Orders Table\n- Lifecycle timestamps (order_placed_at, procurement_completed_at, etc.)\n- Calculated analytics fields (procurement_time, processing_time, etc.)\n- SLA fields (sla_breach, breached_stage)\n- Bottleneck field (bottleneck_stage)\n\n### Products, Suppliers, Users Tables\n- Standard entity management tables\n\n## Development\n\n### Testing\n```bash\npytest\n```\n\n### Generate Sample
